题目概述

GLITCH 是一台 Easy 难度的 Linux 渗透靶机，仅开放 80 端口（Node.js）。攻击路径从 JS 源码发现隐藏 API 开始，经过参数爆破、eval 注入实现 RCE，再通过 Firefox 存储的凭证获取高权限用户密码，最终利用 doas 配置实现 root 提权。

💡 原来 this_is_not_real 就是有效的 Cookie 值！之前误以为是假 token 而忽略了。
