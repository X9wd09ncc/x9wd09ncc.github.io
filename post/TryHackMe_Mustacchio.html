题目概述

Mustacchio 是一台 Easy 难度的 Linux 渗透靶机。攻击路径从 Web 信息泄露开始，经过 XXE 注入读取 SSH 私钥、破解私钥密码、SSH 登录，最终通过 SUID 二进制文件的 PATH 劫持实现 root 提权。

攻击路径

1. 端口扫描

端口 服务 说明
22 SSH OpenSSH 7.2p2 Ubuntu
80 HTTP Apache/2.4.18 — 博客网站
8765 HTTP nginx/1.10.3 — 登录管理页面

80 端口是一个简陋的博客网站，仅有几张图片，文章无法访问。8765 端口是一个登录页面。

2. 目录扫描与信息泄露

2.1 目录扫描

对 80 端口进行目录扫描：

路径 状态码 说明
/custom 301 样式目录
/fonts 301 字体目录
/images 301 图片目录
/robots.txt 200 robots 文件
/server-status 403 禁止访问

2.2 数据库备份泄露

访问 /custom/js 目录，发现备份文件 user.bak，下载后以文本方式打开发现文件头为 SQL 格式标识，修改后缀为 .db 使用 SQLite 打开，获取到管理员凭证：

用户名：admin
密码哈希：1868e36a6d2b17d4c2745f1659433a54d4bc5f4b

哈希长度为 40 位，判断为 SHA-1，使用 CrackStation 解密得到明文：bulldog19
