目录扫描得到 /admin，访问是一个登录页面，但是不知道账号密码

但是在 login.js 里面找到了一串 js 代码

漏洞利用

前端认证绕过

前端向 /api/login 发送 POST 请求，后端返回一个字符串

如果返回 Incorrect credentials，提示密码错误

如果返回任何其他内容，前端就会：把这个内容作为 SessionToken 存入 Cookie，直接跳转到 /admin

直接在控制台设置 cookie，然后跳转，成功登录

SSH 密钥破解

将密码保存，赋予 600 权限，再使用 ssh2john 转化类型，最后用 john 破解，得到密码：james13

SSH 登录

密码解密

有一个 .overpass 文件，里面有一串字符编码

rot47 解码得到 James 的密码

Cron 定时任务

查看定时任务，每分钟以 root 从 overpass.thm 下载脚本并执行

hosts 文件劫持

该用户有对 /etc/hosts 的修改权限，修改文件将 overpass.thm 指向攻击机地址

注意写入后会默认在最后，而该文件第三行会将 overpass.thm 解析为本地地址，因此可以用 nano 将其注释掉

反弹 Shell

本地创建 downloads/src/buildscript.sh，并将反弹指令写入里面

同时 python 开启静态服务

攻击机监听端口，等待靶机下载文件就行
