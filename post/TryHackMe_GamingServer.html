环境信息

  • 目标 IP：10.49.149.195
  • 开放端口：22 (SSH), 80 (HTTP)
  • 操作系统：Ubuntu (Apache/2.4.29)

漏洞总结

漏洞类型 描述 利用方式
信息泄露 HTML 注释泄露用户名 获取 SSH 用户名 john
目录遍历 /uploads/ 和 /secret/ 目录可访问 获取密码字典和加密私钥
弱密码 SSH 私钥使用弱密码 letmein 字典爆破获取私钥密码
LXD 提权 用户属于 lxd 组 创建特权容器挂载宿主机文件系统

防御建议

  1. 移除 HTML 注释中的敏感信息
  2. 配置 Apache 禁止目录遍历：
  3. 保护敏感文件：不要将 SSH 私钥放在 Web 可访问目录
  4. 使用强密码：SSH 私钥应使用强密码保护
  5. 限制 LXD 组成员：仅将可信用户添加到 lxd 组
