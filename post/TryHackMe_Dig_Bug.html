题目描述

Oooh, turns out, this 10.49.156.149 machine is also a DNS server! If we could dig into it, I am sure we could find some interesting records! But… it seems weird, this only responds to a special type of request for a givemetheflag.com domain?

Access this challenge by deploying both the vulnerable machine by pressing the green “Start Lab Machine” button located within this task, and the TryHackMe AttackBox by pressing the  “Start AttackBox” button located at the top-right of the page.

Use some common DNS enumeration tools installed on the AttackBox to get the DNS server on 10.49.156.149 to respond with the flag.

翻译

参与此挑战，需同时部署易受攻击的机器和TryHackMe AttackBox。部署易受攻击的机器，请点击此任务中的绿色“启动实验室机器”按钮；部署TryHackMe AttackBox，请点击页面右上角的“启动AttackBox”按钮。

使用AttackBox上安装的一些常见DNS枚举工具，让10.49.156.149上的DNS服务器响应标记。
