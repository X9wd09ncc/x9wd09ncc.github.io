Ninja Skills

题目描述

请回答有关以下文件的问题：

  • 8V2L、bny0、c4ZX、D8B3、FHL1、oiMO、PFbD、rmfX、SRSq、uqyw、v2Vb、X1Uy

解题过程

1. 哪些文件属于 best-group 组？

D8B3、v2Vb

2. 哪些文件包含 IP 地址？

遍历所有目标文件，用正则匹配 IP 格式：

oiMO

3. 哪个文件的 SHA1 哈希值为 9d54da7584015647ba052173b84d45e8007eba94？

c4ZX

4. 哪个文件包含 230 行？

脚本没有输出，该文件在当前服务器上不存在（find 无结果），后通过穷举法确认为 bny0。

5. 哪个文件的所有者 ID 为 502？

X1Uy

6. 哪个文件可以被所有人执行？

8V2L

答案汇总

问题 答案
属于 best-group 组的文件 D8B3、v2Vb
包含 IP 地址的文件 oiMO
SHA1 为指定值的文件 c4ZX
包含 230 行的文件 bny0
所有者 ID 为 502 的文件 X1Uy
所有人可执行的文件 8V2L
