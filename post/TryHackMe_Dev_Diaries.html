题目

我们刚刚上线了一个由自由开发者制作的网站。开发者没有向我们提供源代码，而且之后就杳无音信了。

尽管如此，网站的开发过程和早期版本的痕迹可能仍然存在于网上。

您目前只能从网站的主域名开始查找：marvenly.com

请回答以下问题

网站开发版本托管在哪个子域名下？

这位开发者的GitHub用户名是什么？

开发者的邮箱地址是什么？

开发者在提交历史记录中提到删除源代码的原因是什么？

隐藏标志的值是多少？

用 Subdomain Finder 扫描，发现了两个子域名。其中一个是 uat-testing.marvenly.com ，但是无法访问。
使用互联网快照馆查看快照
https://web.archive.org/web/20260318003731/https://uat-testing.marvenly.com/
在底部得到了一个用户名

github 搜索该用户名，有一个仓库，仓库里面没什么内容，查看该仓库的提交历史，有四个，其中第二个看起来像是删除源代码的原因

点击第三个，然后在 url 后面添加 .patch，得到了邮箱地址以及flag
