目录扫描

目录扫描得到 /content，访问是一个由 SweetRice CMS 构建的网站，但是没有其他信息

继续目录扫描，首先得到 /content/as 目录，访问是一个登录页面，但是不知道密码，搜索相关漏洞，存在任意密码修改，但是不知道相关邮箱，只好作罢

同样还有 /content/inc 目录，是一个文件目录，其中 /content/inc/lastest.txt 得知 cms 版本为 1.5.1

还有一个 /content/inc/mysql_backup，里面有一个 sql 文件，下载后在里面找到账号密码

密码 md5 解密得到 Password123

后台登录

文件上传

在后台找到一个文件上传的地方，上传单文件 php shell 被拦截，修改后缀为 php5 成功上传

反弹 Shell

权限提升

Sudo 提权

执行 sudo -l，可以无密码以 root 权限执行 /usr/bin/perl /home/itguy/backup.pl

修改脚本提权

查看 backup.pl 文件以及权限，拥有执行权限但是没有修改权限

该命令会运行另外一个文件

查看 /etc/copy.sh 权限，可以进行修改

将提权语句写入该文件
