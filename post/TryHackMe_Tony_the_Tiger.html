页面分析

访问 80 端口，是一个博客网站，只有两篇文章

在其中一个文章中发现作者在图片里面藏了信息

刚好下面就有一个图片，但是里面什么信息都没有

而在另一篇文章也有一个图片

漏洞利用

图片隐写

下载后，使用 strings 命令得到 flag

Java 反序列化

根据提示可知，该网站存在 CVE-2015-7501 反序列化漏洞

下载利用工具 https://github.com/frohoff/ysoserial/releases/latest/download/ysoserial-all.jar

攻击机监听端口，执行下面的命令尝试反弹 shell

成功反弹，输入 id 有回显，但是其他命令则没有

利用 python -c 'import pty; pty.spawn("/bin/bash")' 升级成完整交互式 Shell

SSH 密码发现

该目录下有一个 note 文件，查看里面有一个密码，结合前面开放的端口，很有可能是 ssh 的密码

SSH 登录

使用密码 likeaboss，成功登录

权限提升

Find 提权

sudo -l 发现可以无密码 root 执行 /usr/bin/find

find 有 -exec 参数，可以执行任意命令，用 find 启动 /bin/bash 直接提权到 root

Root Flag

查看 /root/root.txt，是一串 base64 编码

解码后得到一串 hash 值

https://hashes.com/en/decrypt/hash 网站解密

总结

知识点

  • 图片隐写：strings 命令
  • Java 反序列化漏洞（CVE-2015-7501）
  • Find sudo 提权：-exec 参数执行任意命令
