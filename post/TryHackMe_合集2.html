TryHackMe 合集（二）

本合集包含多个 TryHackMe Easy 级别挑战。

Brooklyn Nine Nine

This room is aimed for beginner level hackers but anyone can try to hack this box. There are two main intended ways to root the box.

信息收集

端口扫描，开启了 22，21，80 端口

访问网站，f12 得到提示

漏洞利用

访问图片地址下载图片 /brooklyn99.jpg

用 steghide 查看信息，需要密码

开启了 ftp 服务，尝试匿名登录，下载 txt 文件

里面说 jake 用户使用了弱密码

使用 hydra 破解，得到密码 987654321

使用 ssh 登录成功

权限提升

sudo -l 得到 less 命令可以无密码 root 执行

sudo less 查看 /etc/profile，输入 !bash 然后回车就得到了 root 权限

Lo-Fi

Want to hear some lo-fi beats, to relax or study to? We’ve got you covered!

信息收集

扫描端口得到 80，22 端口

参数使用了 page，可能存在目录遍历

漏洞利用

直接输入多级目录查看 /etc/passwd 验证是否存在目录遍历

遍历 flag.txt

Fowsniff CTF

Hack this machine and get the flag. There are lots of hints along the way and is perfect for beginners!

信息收集

端口扫描

访问，页面说出现了数据泄露

漏洞利用

谷歌搜索该公司，在 github 找到了泄露的数据

在 https://hashes.com/en/decrypt/hash 解密

利用 msf 进行 pop3 爆破

利用 nc 登录并查看邮件

得到了 ssh 密码和账号

baksteen
S1ck3nBluff+secureshell

成功登录

权限提升

寻找该组成员可以执行的文件，有一个 /opt/cube/cube.sh 文件

在该文件中植入反弹命令

重新登录 ssh，本机监听端口，成功得到 root 权限

Smag Grotto

Follow the yellow brick road.

信息收集

端口扫描，开启了 22，80 端口

目录扫描得到 /mail，访问可以下载一个 pcap 文件

漏洞利用

打开文件得到一个网站的账号密码

development.smag.thm/login.php
username=helpdesk
password=cH4nG3M3_n0w

修改 hosts 文件再登录

进去是一个命令执行的输入框，尝试反弹 shell

权限提升

上传 linpeas.sh 来查看有无利用点

发现可以将自己的公钥替换它的公钥

用自己的公钥来替换

ssh 登录，不需要密码

apt-get 有无密码权限，利用 -o 调用外部命令提权

RootMe

A ctf for beginners, can you root me?

信息收集

端口扫描，开启了 22，80 端口

扫目录得到一个 /panel，访问是一个文件上传页面

但是不能上传 php

漏洞利用

猜测是黑名单，修改后缀为 php5 成功上传

权限提升

查找带有 SUID 权限的文件，找到一个 python2

用 mkfifo 反弹 shell

因为 python2.7 有 SUID root 权限，用 os.setuid(0) 提权

Jax sucks alot………….

In JavaScript everything is a terrible mistake.

信息收集

端口扫描，得到 22，80 端口

访问网站，有一个输入框

输入信息没反应，根据题目描述，控制台发现 js 代码

漏洞利用

burpsuite 抓包，session 是 base64 编码

解码后就是输入框输入的数据

构造 poc 并编码

用 get 发包，并构造 cookie，成功反弹

权限提升

执行 sudo -l，利用 /usr/bin/npm 提权
