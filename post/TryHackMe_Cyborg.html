borg 命令可以通过 apt install borg 下载

查看该脚本，接受一个 -c 参数，将其赋值给 command 变量

然后执行 cmd=$($command)，即以 root 权限执行任意命令，最后输出结果

alex@ubuntu:~$ cat /etc/mp3backups/backup.sh
#!/bin/bash

sudo find / -name "*.mp3" | sudo tee /etc/mp3backups/backed_up_files.txt


input="/etc/mp3backups/backed_up_files.txt"


while getopts c: flag
do
        case "${flag}" in 
                c) command=${OPTARG};;
        esac
done


# ... 中间是备份代码 ...

cmd=$($command)
echo $cmd

直接提权，成功，但是执行命令却没有输出结果

这是因为虽然通过 -c 参数可以提权，但无法获得一个交互式 shell，这是因为脚本执行完命令后就退出了
