Cyborg

A box involving encrypted archives, source code analysis and more.

目录扫描

访问网站是一个 apache 默认页面，目录扫描得到 /etc 和 /admin

访问 /etc，是一个文件目录，在 /etc/squid/passwd 发现密码

这是一个 Apache MD5 (apr1) 格式的密码哈希

music_archive:$apr1$BpZ.Q.1m$F0qqPwHSOG50URuOVQTTn。

漏洞利用

密码破解

使用 john 破解，得到明文：squidward

使用该账号密码登录 ssh 失败

Borg 备份提取

继续访问 /admin，是一个很简单的页面，但是有一个下载地址，下载得到一个压缩包

解压

查看 README，告诉我们这是一个 Borg 备份仓库，还给了项目地址

访问该仓库地址，阅读它的相关文档，可以使用 borg extract /path/to/repo::my-files 命令来提取归档文件

borg 命令可以通过 apt install borg 下载

/path/to/repo 就是刚才解压的路径 home/field/dev/final_archive/

my-files 应该是 music_archive，而且提取需要密码，前面和密码相关的也只有这个了

提取后 home 目录下多了一个 alex 文件夹

获取登录凭证

在 /home/alex/Documents 找到了一个登录凭证

SSH 登录

使用该凭证成功以 alex 用户登录 ssh

权限提升

Sudo 提权

sudo -l 发现可以以 root 执行 /etc/mp3backups/backup.sh，同时还有读的权限

脚本分析

查看该脚本，接受一个 -c 参数，将其赋值给 command 变量

然后执行 cmd=$($command)，即以 root 权限执行任意命令，最后输出结果

提权获取 Flag

直接提权，成功，但是执行命令却没有输出结果

这是因为虽然通过 -c 参数可以提权，但无法获得一个交互式 shell，这是因为脚本执行完命令后就退出了

那就直接通过命令方式读取 flag

总结

知识点

  • Borg 备份工具使用
  • MD5 哈希破解
  • Sudo 提权：利用脚本的 -c 参数执行任意命令
