TryHackMe 合集（一）

本合集包含多个 TryHackMe Easy 级别挑战。

Neighbour

Check out our new cloud service, Authentication Anywhere. Can you find other user’s secrets?

进入靶场，是一个登录页面

根据提示 ctrl+u

使用 guest/guest 登录，再次 ctrl+u

注意到 url 后面的参数是 user=guest

将其改为 admin

Anonforce

boot2root machine for FIT and bsides guatemala CTF

信息收集

扫描端口

漏洞利用

用 ftp 匿名登录

在 home/melodias 下找到 user.txt，get 下载到本地

又在 notread 目录下找到两个文件，下载下来

private.asc 是一段 pgp 私钥，backup.pgp 是一段加密字符

开始破解，得到密码 xbox360

使用密码 xbox360 导入私钥

解密 backup.pgp

得到了 /etc/shadow 哈希文件，里面有 root 密码哈希

root:$6$07nYFaYf$F4VMaegmz7dKjsTukBLh6cP01iMmL7CiQDt1ycIm6a.bsOIBp0DwXVb9XI2EtULXJzBtaMZMNd2tV4uob5

破解密码

得到密码 hikari，ssh 登录

Ignite

A new start-up has a few issues with their web server.

信息收集

使用了 Fuel CMS，版本是 1.4

谷歌搜索该版本存在无认证 rce 漏洞 CVE-2018-16763

漏洞利用

验证漏洞

payload：/fuel/pages/select/?filter=%27%2Bpi(print(%24a%3D%27system%27))%2B%24a(%27id%27)%2B%27

kali 搜索相关漏洞并下载 47138.py

使用 python2 执行，成功反弹

在 home/www-data 下找到 flag.txt

权限提升

在网站首页提到了一个数据库文件

查看该数据库文件，给了 root 账号的密码

使用 script 创建一个伪终端

获取 root.txt

Pickle Rick

A Rick and Morty CTF. Help turn Rick back into a human!

信息收集

进入网站，在控制台得到用户名 R1ckRul3s

目录扫描

访问 /robots.txt，只有一串字符串 Wubbalubbadubdub，猜测可能是密码

漏洞利用

访问 /login.php

用户名：R1ckRul3s
密码：Wubbalubbadubdub

成功登录，进去后是一个可以命令执行的输入框

尝试 cat Sup3rS3cretPickl3Ingred.txt，但是命令被禁止了

使用 tac 成功得到第一个 flag

查看 clue.txt，显示 Look around the file system for the other ingredient

输入 ls -l /home/rick，查看该文件，成功得到 flag

权限提升

输入 sudo -l，可以无密码以 root 执行任意命令

输入 sudo tac /root/3rd.txt 得到最后一个 flag

mKingdom

Beginner-friendly box inspired by a certain mustache man.

信息收集

扫描端口

访问 85 端口，就一个鬼脸

目录扫描

漏洞利用

访问 /app，使用了 concrete5 cms

登录页面，使用弱密码 admin/password 成功登录

进入后台，有可进行文件上传的地方

设置里面可以设置上传的文件类型，添加上 php

上传木马文件

权限提升

蚁剑连接，知道了 toad 用户的密码

上传单文件 PHP shell，使用 python 反弹 shell

使用刚才得到的密码登录 toad

执行 env 命令，发现密码字段，base64 解码得到 ikaTeNTANtES

使用该密码登录 mario 账户

修改 hosts 文件，本地创建 counter.sh 写入反弹指令，python 开启 85 端口

等待一会成功反弹

Dav

boot2root machine for FIT and bsides guatemala CTF

信息收集

扫描端口，只开放了 80 端口，进去是一个阿帕奇默认页面

使用 gobuister 扫目录，得到目录 /webdav

访问该地址弹出登录框

webdav 是阿帕奇自带的一个模块，使用 wammp/xammp 成功登录

漏洞利用

创建 shell.php，里面写入反弹指令

使用 cadaver 连接，然后使用 put 上传后门

本机监听端口，再去网页端访问 /webdav/shell.php

成功反弹

权限提升

sudo -l 发现可以无密码 root 执行 /bin/cat

那就直接查看 root.txt

TakeOver

This challenge revolves around subdomain enumeration.

访问网站会重定向到 futurevera.thm，修改 hosts 文件即可

子域名扫描

但是无法访问

查看证书得到一个新的地址

curl 访问，得到 flag

Couch

Hack into a vulnerable database server that collects and stores data in JSON-based document formats.

信息收集

端口扫描，其中一个是 couchdb 数据库

访问 /_utils 查看 web 管理器

漏洞利用

访问 /_all_dbs 查看所有数据库

在 secret 数据库里面找到一个账号密码

atena
t4qfzcc4qN##

使用 ssh 登录

权限提升

查看 .bash_history，有一条 docker 命令

运行该命令得到 root 权限
