TryHackMe_h4cked

任务一

用 wireshark 打开 pcap 文件，追踪 tcp 流，可以看到攻击者正在尝试登录 ftp 服务，使用的用户名是 jenny，并且在第 7 个流中可以看到登录成功，密码是 password123

第 16 个流中可以看到攻击者执行了 pwd 命令，当前目录为 /var/www/html，并且上传了 shell.php 文件

第 18 个流就是 shell.php 的内容，在注释里面有一个地址

第 20 个流就是攻击者拿到 shell 以及提权到 root 的流程
第7行，攻击者拿到 shell 后执行了 whoami 命令
第39行，攻击者使用 python 创建了一个伪终端
第40行，显示的主机名是wir3
第54行，攻击者通过执行sudo su拿到了 root 权限
第64行，攻击者使用 git 下载了Reptile

尝试访问这个 github 项目，但是目前这个仓库被禁用了
谷歌搜索这个项目，发现谷歌给出的结果里面大多都有 Rootkit，也就是肉鸡，也就是这个项目会在服务器里面留下一个肉鸡后门

任务二
现在靶场给出了服务器地址，并提示我们 flag 位于/root/Reptile 目录下，且攻击者修改了 ftp 密码
先进行端口扫描，确认 ftp 服务的端口

使用 Hydra 爆破，得到密码 987654321

修改原本的 shell.php，只需要就 ip 地址修改就行

然后登录 ftp，上传修改好的文件，并赋予 777 权限

kali 监听 4444 端口，然后用 curl 访问 shell1.php

创建伪终端，切换至 jenny 并提权
