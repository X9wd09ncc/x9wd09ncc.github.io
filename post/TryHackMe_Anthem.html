Anthem

Exploit a Windows machine in this beginner level challenge.

信息收集

端口扫描

序号 Host地址 端口 协议 Uri地址 标题 Banner 状态码

目录扫描

得到以下目录：

  • /archive
  • /authors
  • /blog
  • /categories
  • /install → 重定向到 /umbraco
  • /robots.txt
  • /rss
  • /search
  • /SiteMap
  • /tags
  • /umbraco

访问 /robots.txt，第一行有一串字符 UmbracoIsTheBest!，可能是密码

还有 /umbraco 目录，因此使用的 CMS 就是 Umbraco

漏洞利用

信息收集

访问网站首页，在输入框里面看到了一个 THM 开头的字符串，F12 查看，得到了其中一个 flag

进入其中一篇文章，同样在控制台找到了一个 flag

有一个邮箱 JD@anthem.com，作者名是蓝色的，可以点击

点击作者名来到作者页面，找到了另一个 flag

来到另一篇文章，在控制台找到了最后一个 flag

这篇文章还有一首诗，描述的是网站管理员

获取管理员凭据

谷歌搜索这首诗，进入维基页面，标题就是该网站的管理员名字 Solomon Grundy

至于邮箱，前面有一个作者 Jane Doe，他的邮箱是名字简写 JD 开头

因此猜测该管理员的邮箱也是名字简写 SG 开头，即 SG@anthem.com

RDP 登录

该网站还开了 RDP 服务，用 windows 的 mstsc 来连接

用户名：SG
密码：UmbracoIsTheBest!

进去后在桌面就有一个 user 文本，打开拿到第一个 flag

总结

知识点

  • 信息收集：robots.txt、源码注释、作者页面
  • CMS 识别：Umbraco
  • RDP 远程桌面连接

任务答案

任务1 网站分析

问题 答案
Web服务器的端口是什么？ 80
远程桌面服务使用哪个端口？ 3389
网络爬虫会检查哪些页面，其中可能包含哪些密码？ UmbracoIsTheBest!
该网站使用的是什么内容管理系统（CMS）？ Umbraco
该网站的域名是什么？ anthem.com
管理员的名字是什么？ Solomon Grundy
我们能找到管理员的电子邮件地址吗？ SG@anthem.com
