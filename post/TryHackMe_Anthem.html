访问 /robots.txt，第一行有一串字符 UmbracoIsTheBest!，可能是密码

还有 /umbraco 目录，因此使用的 CMS 就是 umbraco

访问网站首页，在输入框里面看到了一个 THM 开头的字符串，F12 查看，得到了其中一个 flag

进入其中一篇文章，同样在在控制台找到了一个 flag

有一个邮箱 JD@anthem.com ，作者名是蓝色的，可以点击

点击作者名来到作者页面，找到了另一个 flag

来到另一篇文章，在控制台找到了最后一个 flag

这篇文章还有一首诗，描述的是网站管理员

谷歌搜索这首诗，进入维基页面，标题就是该网站的管理员名字

至于邮箱，前面有一个作者 Jane Doe，他的邮箱是名字简写 JD 开头

因此猜测该管理员的邮箱也是名字简写 SG 开头

该网站还开了 RDP 服务，用 windows 的 mstsc 来连接

用户名经过尝试确认是 SG，密码则是 UmbracoIsTheBest!

进去后在桌面就有一个 user 文本，打开拿到第一个 flag

但是无奈这个连接太卡了，我光是打开这个文本就卡了好久，后面没办法进行下去了

任务1网站分析

这项任务需要你注意细节，找到“城堡的钥匙”。

这个房间是为初学者设计的，不过，欢迎所有人来体验！

欣赏国歌。

在这个房间里，你不需要暴力破解任何登录页面。只需要你常用的浏览器和远程桌面即可。

请给设备最多 5 分钟的启动和配置时间。

请回答以下问题

Web服务器的端口是什么？
80

远程桌面服务使用哪个端口？
3389

网络爬虫会检查哪些页面，其中可能包含哪些密码？
UmbracoIsTheBest!

该网站使用的是什么内容管理系统（CMS）？
Umbraco

该网站的域名是什么？
anthem.com

管理员的名字是什么？
Solomon Grundy

我们能找到管理员的电子邮件地址吗？
SG@anthem.com

任务2 找出 flag

我们敬爱的管理员留下了一些标记，我们需要收集这些标记才能继续进行下一个任务。

请回答以下问题

flag 1 是什么？
THM{L0L_WH0_US3S_M3T4}

flag 2 是什么？
THM{G!T_G00D}

flag 3 是什么？
THM{L0L_WH0_D15}

flag 4 是什么？
