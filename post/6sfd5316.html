Library

题目描述：boot2root 机器适用于 FIT 和 bsides 危地马拉 CTF

端口扫描

访问 80 端口，是一个博客页面，其他地方点击无响应，出现了一个用户名：meliodas

没东西那就用 gobuster 扫描目录

得到目录 robots.txt

访问 /robots.txt

端口有开放 22 端口，80 端口发现用户名，/robots.txt 提示 rokyou 字典，于是尝试用 hydra 破解 ssh 密码

成功破解得到密码 iloveyou1

使用用户名 meliodas 和密码 iloveyou1 登录 ssh 服务

查找 user.txt，得到第一个 flag

使用 sudo -l 查找可无密码执行 root 权限的命令

得到命令 /usr/bin/python* /home/meliodas/bak.py

查看 bak.py，只是一个备份脚本，把 /var/www/html 打包成 /var/backups/website.zip

并没有可以提权的点

尝试修改 bak.py ，但是没有权限

后面注意到 bak.py 引用了 zipfile，于是可以构造一个恶意的 zipfile.py，在运行 bak.py 的时候他就会优先调用同目录下的 zipfile.py，从而达到提权的目的
