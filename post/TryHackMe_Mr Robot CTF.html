访问网站，是一个很高级的页面，但是输入这些命令后都是一些看不懂的英文视频和图片

![](../image/TryHackMe/medium/Mr_Robot_CTF/TryHackMe_Mr Robot CTF_20260618193904.png)

目录扫描，这么多 wp 开头的文件，看来用了 wordpress
目录很多，有用的也就几个

访问 robots.txt，有两个文件

fsocity.dic 看起来像是一个字典文件,字典很大，也有很多重复内容，排序并去重

访问 login，是 wordpress 的登录页面
burpsuite抓包对用户名进行爆破，密码随意
在密码错误的情况下， 用户名 Elliot 的响应长度不一样，猜测这个就是用户名

![](../image/TryHackMe/medium/Mr_Robot_CTF/TryHackMe_Mr Robot CTF_20260618195059.png)

接下来爆破密码

![](../image/TryHackMe/medium/Mr_Robot_CTF/TryHackMe_Mr Robot CTF_20260618200247.png)

得到登录凭据

用户名：Elliot
密码 ： ER28-0625

进入后台，找到 Appearance - Editor，就选择第一个 404 模板文件，将里面的内容替换为一个单文件 php shell

![](../image/TryHackMe/medium/Mr_Robot_CTF/TryHackMe_Mr Robot CTF_20260618201348.png)

接下来就要访问这个 php 文件，当我修改这个文件时，url 地址是

http://10.49.181.52/wp-admin/theme-editor.php?file=404.php&theme=twentyfifteen&scrollto=0

按照 wordpress 的规则，该文件的访问地址是

http://10.49.181.52/wp-content/themes/twentyfifteen/404.php

![[../image/TryHackMe/medium/Mr_Robot_CTF/TryHackMe_Mr Robot CTF_20260618201431.png]]
