三个核心路由

路由 功能
/cloud GET 显示下载页面；POST 根据 download 参数返回文件
/debug 获取 debug 和 password 参数，校验非法字符后比对加密密码
/debugresult 检查 IP 合法性，从 session 中获取 debug 内容并渲染模板

关键漏洞点

/debugresult 路由中，debug 内容直接来自用户输入，使用 render_template_string 渲染且未进行过滤 → **SSTI (服务端模板注入)**。
