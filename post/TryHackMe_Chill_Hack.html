命令执行绕过

反弹 Shell

直接反弹 shell

kali 监听，成功反弹

脚本提权

在 /home/apaar 目录下找到一个 www-data 可以执行的文件 .helpline.sh

由于 $msg 被直接执行，因此可以传入任意系统命令

使用 echo -e "aaa\nsudo -l" | ./.helpline.sh 来执行 sudo -l 命令

www-data 可以无密码以 apaar 用户身份执行 /home/apaar/.helpline.sh

先使用 script 创建一个伪终端，然后以 apaar 身份执行文件，成功切换用户，并在 local.txt 里面得到 flag

隐写分析

对 apaar 尝试各种方法提权都不行，看 wp 关键点还是在 www-data 这里，进入到 /var/www/files 目录

有一个 hacker.php，有一个提示，大概意思是在黑暗中会找到答案，而刚好上面有一张图片

wget 下载到本地，使用 steghide 得到隐写的 zip，但是解压需要密码

Zip 密码破解

使用 john 破解得到密码：pass1word

解压后在里面的 php 文件里面又得到一个密码，解码结果为

!d0ntKn0wmYp@ssw0rd

SSH 登录

猜测是 ssh 的密码，前面得到了四个用户，一个一个尝试，最后以 anurodh 用户身份登录

并且执行 id 后发现该用户在 docker 组里面
