🧠 六、技术总结

6.1 核心难点

难点 解决方案
关键词过滤 (UNION, SELECT) 大小写混合绕过：uniON, seLect
注释符过滤 (--, /*) 使用字符串拼接 `
尾部引号闭合 拼接字符串 'a 使尾部 ' 成为闭合符

6.2 SQLite 特性利用

  • **sqlite_master**：SQLite 系统表，存储所有表的结构信息
  • **group_concat()**：聚合函数，将多行合并为一行
  • || 运算符：SQLite 字符串拼接

🛡️ 七、防御建议

  1. 使用参数化查询（Prepared Statements）
  2. 输入验证：白名单校验，而非黑名单过滤
  3. 最小权限原则：数据库用户权限最小化
  4. 错误信息隐藏：不向用户暴露数据库错误详情
  5. WAF 部署：检测并拦截 SQL 注入攻击
